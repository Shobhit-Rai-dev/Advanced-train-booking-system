# Advanced Railway Ticket Booking System (C)

A comprehensive **C-based console application** that simulates an **IRCTC-like train reservation system** with:
- Admin control  
- Passenger management  
- File handling  
- Waitlist system  
- Meal options  
- E-wallet features  

This project demonstrates **structured programming, file handling, arrays, structures, and modular design in C**,  
making it ideal for **academic submission or GitHub portfolio display**.

---

##  Features

###  Passenger Functions
- Book tickets (with automatic seat allotment and fare calculation)
- Cancel tickets (auto reassigns to waitlist)
- Search passenger by name
- Print ticket summary
- View waitlist
- Provide feedback/complaints
- View available seats
- View fare chart
- Choose meal plan
- Manage e-wallet balance
- Add emergency contact information

### Admin Panel
Accessible with password: **`1234`**
- Add new trains
- Delete existing trains
- View all trains
- View all booked tickets
- View revenue report (total fare collected)
- View passenger feedback
